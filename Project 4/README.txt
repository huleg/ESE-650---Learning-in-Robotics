Steps to run the code

1) First run reFormatData.m to generate the a single file containing the lidar scans and head angles along with their time stamps. Change lines 5,6 and 19 according to the file names.

2) Run main.m to start slam. Change lines 9, 12, 16 and 108 according to the file names

3) Turn texture mapping on or off by setting it to 1 or 0 respectively.
